## Installation 💻

```lua
npm install -g @expressots/cli # Eventually
```

## Local usage 🏠

```bash
npm install
npm run start
```

### Help message 🆘

```bash
npm run start --help
```

### Create a new project ⚙️

```bash
npm run start new project
```
